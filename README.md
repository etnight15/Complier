# Лабораторная работа №1

## Название и цель лабораторной работы
Название: Разработка пользовательского интерфейса (GUI) для языкового процессора.
Цель: Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

## Сведения об авторе
Марков Д.Д., АВТ-314

## Описание проекта
Оконное приложение представляет собой текстовый редактор с графическим интерфейсом, реализующий базовые операции работы с текстовыми файлами. Включает главное меню, панель инструментов, область редактирования и область вывода результатов. Приложение разработано с учетом дальнейшего расширения функционала для создания языкового процессора.

## Используемые технологии
Python 3.9+, PyQt6
Среда разработки: VS Code

## Инструкция по сборке и запуску
### Установка зависимостей
```bash
# Создание виртуального окружения (рекомендуется)
python -m venv venv

# Активация виртуального окружения
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```
## Описание интерфейса и функций (руководство пользователя)
При нажатии на пункт меню "Файл" откроется дополнительное меню, где можно выполнить следующие команды: создать, открыть, сохранить файл, выход.


При нажатии на пункт меню "Правка" откроется дополнительное меню, где можно выполнить различные действия с кодом.


При нажатии на пункт меню "Текст" откроется дополнительное меню, где можно посмотреть все для текста кода.


При нажатии на пункт меню "Пуск", приложение выполнит код и выведет в панели вывода ошибки, если они там есть.


При нажатии на пункт меню "Справка" откроется дополнительное меню, где можно ознакомиться со сведениями о программе.
